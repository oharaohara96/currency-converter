Conversor de Moedas - Triple Peaks
Este é um projeto de conversor de moedas em tempo real desenvolvido como parte dos meus estudos de Front-End. O foco principal foi a manipulação do DOM com JavaScript, lógica de condições e estilização moderna com CSS.

🚀 Tecnologias Utilizadas
HTML5: Estruturação semântica do projeto.

CSS3: Estilização com Flexbox, efeitos de hover, box-shadow e responsividade básica.

JavaScript (ES6): Lógica de conversão, manipulação de eventos (click e change) e formatação de moedas.

⚙️ Funcionalidades
[x] Conversão de Real (BRL) para Dólar (USD).

[x] Conversão de Real (BRL) para Euro (EUR).

[x] Troca dinâmica de bandeiras e nomes das moedas ao selecionar o destino.

[x] Validação de campos (evita converter se o valor estiver vazio ou se as moedas forem iguais).

[x] Interface reativa que atualiza o símbolo da moeda instantaneamente.

🎨 Visual do Projeto
O projeto conta com um design moderno e minimalista, utilizando:

Fundo em escala de cinza (Dark Mode).

Botões com feedbacks visuais (interatividade).

Cartões centralizados com Flexbox.
